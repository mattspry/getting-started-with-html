The World Wide Web
The web is a collection of HTML (HyperText Markup Language) documents, and currently contains over 30bn pages. Pages are navigated using hyperlinks.

Major web elements
Ther are four major elements:
1 - web-user - you!
2 - a computer with web-browser e.g. Chrome, Firefox
3 - the internet
4 - servers

When a user accesses the web using their pc and browser, they are sending a request via the web to access a webpage stored on a server. HTTP is used to access the webpage that is stored on the server.

HTML
HTML consists of:
text content - what you see
markup - what the html looks like
references to other html documents - such as videos and images
links to other pages - hyperlinks

HTML markup

HTML markup is the syntax used to structure content in a HTML document. It consists of elements, in which the content is prefixed by opening tags and suffixed by closing tags - together these tags determine how your content is presented. Tags can be nested to apply multiple factors.

Common tags allow you to apply bold or italic formatting to your content, add line breaks or structure your content in paragraphs.

Tags also allow you to include images, links to other websites or videos in your content

Most tags require an opening and closing tag - however some tags are void tags, and require no closing tag as they are self enclosed e.g. the image tag that brings an image into your html, inserting a line break.

HTML defaults to inline presentation i.e. a continuous stream of content. Certain tags create blocks within your content, allowing them to be presented better and adjusted discretely by CSS or JS.

Span and div elements define blocks of content in either inline (span) or block (div) states.

HTML documents
HTML markup is structured into HTML documents that allow web browsers to interpret and present content correctly. HTML documents are structured into head and body sections. Head sections contain metedata e.g. CSS and JS links, webpage title, whilst the body section contains the webpage content.
